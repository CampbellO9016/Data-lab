# **Shoe Sales Organizer** 
### The Shoe Class 
The shoe class is used to organize shoes. It is used to create a new shoe with its price, color, type, and brand.
### The Shoe Sale Class 
Using this shoe class we can make a new sale of shoes that includes the date, the country, where it was purchased, how many of that show were sold, and the revenue.
### The Main 
The main code of this is very user-friendly, it gives you a way to make a shoe and create a shoe sale. It allows you to input a large amount of data of shoe sales to be stored.
